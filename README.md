Bu script sizin türk sohbet oyunlarında yerdeki paralı bulmakla uğraşmamanız için pstrxion.inc tarafından hazırlanmıştır

# Çalıştırma
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/PSTRXION/neon-cash-finder/refs/heads/main/neon-cf.lua"))()
```
# Nasıl Kullanılır?
Neon Cash Finder scriptini çalıştırdığınız zaman size şifre soracaktır şifreyi 'ŞİFRE' başlığının altından nasıl alacağınızı öğrenebilirsiniz.
Neon Cash Finder a giriş yaptığınızda sol üst küşeye yuvarlak üzerinde 'Neon' yazılı bir buton ve önünüze asıl menü gelecektir.
Yuvarlak buton sayensinde büyük menüyü gizleyebilirsiniz.
